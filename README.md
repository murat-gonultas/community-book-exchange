# Community Book Exchange

Closed-community Android app for tracking physical books inside a private community.

## Goals
- Track who owns a book
- Track who currently holds a book
- Support reservation, loan, return, gift, and donation flows
- Keep a searchable and structured inventory
- Run with zero hosting cost on an always-on old laptop

## Planned stack
- Flutter Android app
- Spring Boot backend
- SQLite database
- Cloudflare Tunnel
